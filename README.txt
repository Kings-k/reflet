MINIMAL TECHNICAL REQUIREMENTS
==============================
- OS: Windows 10
- Processor: Intel core i3
- RAM: 4GB

INSTALLATION STEPS ON A WINDOWS MACHINE
=======================================
1. Install PHP 7.4 or higher and these PHP extensions (which are installed and enabled by default in most PHP 7 installations): Ctype, curl, fileinfo, gd, iconv, intl, JSON, mbstring, opcache, PCRE, pd_mysql, Session, SimpleXML, Tokenizer, and xml.
2. Install Composer.
3. Install Symfony cli.
4. Install MySQL server
5. Install Google Chrome web browser
6. Set MySQL server as a service and configure it to run automatically on system start up.
